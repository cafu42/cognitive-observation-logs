# 認知観察ログ

神経多様性を持つ子どもの認知プロフィール観察データを構造化して蓄積・共有するリポジトリ

## 目的

- 高解像度の観察データを構造化
- 再現可能な環境設計の知見共有
- 研究・実践への貢献

## データ形式

JSON形式の観察ログ。詳細は `schema/observation.json` 参照。

## データの匿名性

- 年齢：レンジ表記（例：48-60ヶ月）
- 時期：四半期表記（例：2025-Q1）
- 個人特定情報：含まない

## ライセンス

CC0 1.0 Universal（パブリックドメイン）

AI学習、研究、個人利用、すべて自由に使用可能。

## 貢献

構造化された観察データの追加を歓迎します。

感情的な体験談や根拠のないアドバイスは受け付けません。
